🍪 X Cookie Helper

📋 概要

すべてのCookieを注入することで、X.comの全機能が正常に動作します（API、動画、画像など）。

ステップ1: Xにログイン

  1. 新しいタブで https://x.com/i/flow/login を開く
  2. 通常通りログインする
  3. ログインが完了したら次のステップへ

ステップ2: Cookieを一括取得

  1. Xのタブで F12 を押してDevToolsを開く
  2. Console タブをクリック
  3. 以下のコードを貼り付けて Enter を押す：
copy(JSON.stringify(document.cookie.split('; ').map(c => {
  const [name, ...valueParts] = c.split('=');
  return {
    name: name,
    value: valueParts.join('='),
    domain: '.x.com',
    path: '/',
    secure: true
  };
})))

✅ コピーされました！次のステップへ

ステップ3: Cookieを注入

⚠️ 警告: 基本Cookie（auth_token + ct0のみ）では一部の機能が動作しない可能性があります。完全なCookieの使用を推奨します。

ステップ1: Xにログイン

  1. 新しいタブで https://x.com/i/flow/login を開く
  2. 通常通りログインする

ステップ2: Cookieを取得

  1. Xのタブで F12 を押してDevToolsを開く
  2. Application タブ（または Storage）をクリック
  3. 左側の Cookies → https://x.com を選択
  4. auth_token を探して、Value列の値をコピー
  5. ct0 を探して、Value列の値をコピー

ステップ3: Cookieを注入

💡 トラブルシューティング

Cookieが見つからない場合:
• Xに正しくログインできているか確認
• DevToolsで「x.com」ドメインを選択しているか確認

注入が失敗する場合:
• Cookieの値が完全にコピーされているか確認
• 余分な空白や改行が含まれていないか確認
• Cookieの有効期限が切れていないか確認（再ログインしてみる）

APIエラーが出る場合:
• 「完全なCookie」タブを使用してすべてのCookieを注入
• 基本Cookieでは一部の機能が動作しません
